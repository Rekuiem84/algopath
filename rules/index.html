Rules

Number of players : 1 - 4

Game time : at least 5 minutes for each level

Equipments : the game board, at least one whiteboard marker, a character, a flag, 3 monster tokens, 3 coin tokens, 30 variable tokens, 20 orange action cards, 16 green action cards, 6 blue action cards, 4 red action cards

Setup :

  • Place the board at the center of the table, with a pile of cards for each action (move forward, turn, repeat, attack) next to it.
  • Scan the QR code on the board to access the app, you will enter all the players' names to keep track of the points, and will be asked questions for the quiz.
  • Select a level blueprint on the app and replicate it on the board using the marker, place the character and the flag on the board as well. If you have some monster and coin tokens, place them in their designated squares, if you don't have any, you can draw them using a marker.
  • Choose a game mode on the app : speedrun mode, or greedy mode.

How to play :

  • The player who owns the phone will play first, and the game plays clockwisely (same way a clock hand turns).
  • Select a question :
    • blue questions are easier but only give one action card,
    • red questions are harder but give two action cards.
  • If the player answers correctly to the question, they can draw one card, and one additional card if the question was red.
  • Then, the player can either :
    • end their turn, and the player on their left can start their turn,
    • submit their algorithm to reach the flag
  • If a player successfully reaches the flag, they earn 3 points and the board can be wiped to start a new level.

How to submit the algorithm :

  • The current player gets to choose which way the character faces when starting his algorithm.
  • The current player places their action cards next to the board, from top to bottom, in order, so that the actions make the character reach the flag. The actions are read from top to bottom, from left to right. The current player is free to use as many variable tokens as they need.

How to win :

  • In speedrun mode :
    • Be the first to reach the flag 5 times to win. Once a player reaches the flag, the board is wiped and a new level can be drawn on the board.
  • In greedy mode :
    • Be the first to reach 20 points to win.
    • Points chart :
      • Reaching the flag : 3 points
      • Beating a monster : 2 points
      • Collecting a coin : 1 point
    • Specific actions :
      • To beat a monster, you have to use an attack action card, facing the monster
      • To collection a coin, the character has to walk on the coin tile

Action cards :

  • Orange cards : move N blocks forward, where N is the variable token
  • Green cards : rotate N degrees to the right, where N is the variable token (N can only be 90, 180, or 270)
  • Blue cards : repeat block N times, where N is the variable token
    • the block is the group of action cards under this card, you have to indent your block to the right for it to be repeated (see example below)
  • Red cards : attack the tile in front of the character

Note : players can only submit their algorithm during their turn

Note : the character can never pass a wall, even if it has remaining steps to move. In this case, the character stops moving forward and the next action card is triggered

Tip : turning 270 degrees to the right is the same as turning 90 degrees to the left

About the repeat action card :

This algorithm uses a repeat card, the indented cards will be executed as many times as written the variable token on the repeat card
